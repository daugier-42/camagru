# camagru

> Application web de photos et montages — Projet 42

## Description

Site PHP permettant aux utilisateurs de s'inscrire, se connecter, prendre ou importer des photos via webcam, appliquer des superpositions (overlays) et les partager sur un mur public.

Fonctionnalités : authentification, galerie, likes, commentaires, envoi d'e-mails de confirmation.

## Stack

`PHP` · `HTML` · `CSS` · `JavaScript` · `MySQL`

## Lancement

Configurer la base de données via `config/`, puis servir le dossier avec un serveur PHP :

```bash
php -S localhost:8080
```

## Compétences

`Web` · `Sécurité` · `Sessions` · `Upload de fichiers`
